# Instrunet AI Presentation

This folder contains the source code for the project presentation.

## Compilation Instructions
To generate the PDF from the `.tex` file, use any LaTeX compiler (like `pdflatex`).

```bash
pdflatex presentation.tex
```

## Structure
- `presentation.tex`: The main beamer LaTeX file.
- It uses relative paths to images located in `outputs/` and `outputs/frontend/`.
